We use a number of open-source technologies for running this project. For the curious, here is a curated list:

  • FluxBB as the forum software
  • Grafana for building internal monitoring dashboards
  • InfluxDB for storing time-series data collected by collectd
  • LXC for running multiple services well-separated from each other on the same server
  • MySQL as the forum database
  • collectd for collecting metrics from the system, forum database, webserver, etc
  • git for our VCS
  • nginx is what we use for our webserver
  • reprepro manages our Debian package repository
  • pbuilder builds our packages for i386, amd64 and ARM architectures
  • opentracker as our bittorrent tracker
