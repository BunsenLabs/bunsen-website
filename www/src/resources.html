BunsenLabs is being developed on Github.

Community and communication

The following communication channels are considered ‘official’:

  • Forums
  • IRC channel #bunsenlabs@freenode

As you may know, we had been using the CrunchBang Linux forums in the beginning. Registrations over there are now closed, but there’s much knowledge left for you to harvest.

Pages have also sprung up elsewhere:

  • User Balloon has created an extensive resource in Japanese: BunsenLabs Japan!・ブンゼンラボ ジャパン！, including instructions and translations for a Japanese localization
  • On Google+, there is BunsenLabs en Español
  • DeviantArt group

Core packages and bug reports

Detailed package indices and content listings are available on the ‘repositories’ page.

If you have a bug to report, you may do so either on the forums or, ideally, to the package’s issue tracker. Use the VCS browser links provided below to browse the package sources.
